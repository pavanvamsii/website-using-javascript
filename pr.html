FAQ's

what is js ?

  • JavaScript is the world's most popular programming language.
  • JavaScript is the programming language of the Web.
  • JavaScript is easy to learn.

what is HTML ?

  • Html is a hyper text markup language
  • It is used to design the structure of the web page
  • HTML is not a programming language,it is markup language
  • Everything in HTML we write in the form of tags,an html element usually consists of a start tag(open tag) and end tag(closing tag)

what is CSS ?

  • CSS stands for cascading style sheets,it is a type of style sheet language
  • To make any web page presentable i.e; to add elements such that it appeals to the user ,we use css
  • CSS helps us to control the text color font style the spacing between the paragraphs,sizing of columns layout designing and many more
  • CSS helps web designers to apply different styles on HTML
  • Used with HTML to change the style of web pages and user interface!

Features of javascript

  • javascripting is a client side scripting language
  • a script is a small piece of code
  • javascript is platform independent which can run on any system
